# C3-GRC

CrochetCrochetCrochet (C3) is a SaaS-style company offering Crochet as a Service. Subscribers pay a monthly fee to access pattern libraries, live virtual instruction sessions, and a community forum. C3 stores customer PII (names, emails, payment info), hosts its platform on a cloud provider (AWS), and has a small internal team of 16 people. 

Project Idea Based on: https://artempolynko.com/blog/7-grc-analyst-hands-on-projects/
